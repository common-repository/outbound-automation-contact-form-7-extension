Outbound Automation extension for Contact Form 7

== Description ==
Outbound Automation extension for Contact Form 7 will capture form submits on your WordPress site and upload them to a dynamic List in Outbound Automation.

This is used with [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) and [Outbound Automation](https://marketing.boberdoo.com/) together.  

Dynamic Lists can be created inside the [Outbound Automation](https://marketing.boberdoo.com/) dashboard, or from the [Contact Form 7](https://wordpress.org/plugins/contact-form-7/) page.

[Outbound Automation](https://marketing.boberdoo.com/) is created and maintained by [boberdoo.com](https://www.boberdoo.com)

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go into the Contact Form 7 section of your `wp-admin`
4. Slect the list you would like to use for OA
5. Click the Outboud-Automation Tab and enter you new Listname and API key.

== Use ==

After you have set up Contact From 7 And Outbound Automation in wordpress, any contact form submitted will be sent to list name you entered in Outbound Automation.
